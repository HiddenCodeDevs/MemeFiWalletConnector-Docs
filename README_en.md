# <p align=center>Installation</p>
1. To install, download the archive from the chat (available after purchase)
2. Extract it to a convenient location
3. For Windows:
   
   ```shell
   > cd <your folder path>
   > run.bat
   ```

   ```shell
   $ cd MemeFiWalletConnector
   $ sh run.sh
   ```
